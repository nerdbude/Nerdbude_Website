NixOS 21.05 (Okapi)

Nun benutze ich, wie ich schon mal erwähnt habe (LINK) das beste OS der Welt - NixOS.
Von NixOS gab es nun - wie jedes Jahr - ein Update auf Version 21.05 (Okapi)

Was ist Neu

Es gab Aktuallisierungen in folgenden Bereichen:

Core:
- gcc 9.3.0 -> 10.3.0
- glibc 2.30 -> 2.32
- default linux 5.4 -> 5.10
- mesa 20.1.7 -> 21.0.1

Desktops:
- gnome 3.36 -> 40
- plasma5 5.18.5 -> 5.21.3
- kdaApplications 20.08.1 -> 20.12.3
- cinnamon 4.6 -> 4.8.1

Programmiersprachen und Frameworks:
- "Python optimizations" wurden deaktiviert weil Builds nicht reproducible sind. Können aber händisch aktiviert werden.

Das gesamte changelog gibt es HIER


UPGRADE

Ganz wichtig - Backup machen bevor das Upgrade durchgeführt wird!
Um NixOS ein Upgrade zu verpassen geht man wie folgt vor.
Als erstes solltet ihr checken welchen channel nutzt:


CHANNEL 
nix-channel --list | grep nixos

Danach wird der neue 21.05 channel hinzugefügt:

ADD CHANNEL 
nix-channel --add https://nixos.org/channels/nixos-21.05 nixos

Und zu guter letzt noch das eigentliche Upgrade via:

UPGRADE 
nixos-rebuild switch --upgrade

Tada! - NixOS 21.05 - das beste OS der Welt!
