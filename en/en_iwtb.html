iwtb - colorscheme





If you need new colors in the terminal or in Vim, you can use the IWTB colorset from now on. Especially screwed together by me and found to be good. A colorful set for dark environments with a touch of 80s/90s. Currently you can find the terminal version and the beta for Vim on Github. If you have other tools and want to use the theme there, you are of course free to implement the whole thing. The only requirement is the contribution of your implementation to my Github repo. :)
