** VIM WEB BROWSER **

[PRO] | [20200101] | [0] | [0x17]



Hier gab es schon länger nichts mehr auf der NERDBUDE. Das hat zwei einfache Gründe. Der erste: Ich hatte hier eine kleine Infrastruktur-Apokalypse. Mir ist meine NixOS Instanz hier abgeraucht. Der perfekte Zeitpunkt um mal den neuen grafischen Installer von NixOS zu testen und die configuration.nix und die xmonad.hs aufzuräumen. Der zweite Punkt der hier gerade Zeit verschlingt ist das NB100 Cyberdeck. Dazu gibt's dann hier natürlich auch noch was zu lesen. Dank der ersten Problematik konnte ich aber auch neue Tools finden so z.B. VIMB.

VIMB

Immer weiter getrieben von der suche nach dem möglichst minimalistischen aber dennoch maximal produktivem System, sind die gängigen GUI Webbrowser nicht grade die erste Wahl. So bin ich auf VIMB gestoßen. VIMB ist ein Webbrowser ohne viel Bloat und das tolle daran ist, er beruht komplett auf Vim-Keybindings und wird auch Vim-typisch bedient. VIMB lässt sich also wunderbar in den täglichen Workflow integrieren.

INSTALLATION

VIMB ist in den nix-packages vorhanden, d.h. ich kann sie direkt in die configuration.nix einbinden und per rebuild installieren.

configuration.nix 
environment.systemPackages = with pkgs; [
vimb
...
];
in {


Wie immer nach Änderungen an der configuration.nix braucht es den rebuild: nixos-rebuild switch.
Danach kann VIMB benutzt werden.

USAGE

Entweder ihr startet VIMB per Terminal oder ihr definiert euch einen Shortcut innerhalb der xmonad.hs. Um VIMB per Terminal zu starten reicht folgendes Command:


TERMINAL 
vimb


danach ist der Browser direkt benutzbar. Die Frage ist nur wie?
Ganz einfach. VIMB lässt sich bedienen wie Vim, d.h. bei jeder Interaktion braucht ihr den Vim-typische ":". So kann beispielsweise eine Website mit folgendem Befehl geöffnet werden:

VIMB 
:open https://www.nerdbude.com


Das öffnet euch (in diesem Fall wunderschöne) Website. Nun wollt ihr aber vielleicht doch etwas mehr machen als einzelne Websites aufrufen. Dazu bringt VIMB eine ganzes Set an Commands mit.

VIMB 
:open https://www.nerdbude.com
